Description

This is a sub-class of the AreaCorrect class for the implementation of the Morris, Pannala, Ma, Hrenya contact area correction model.

This model computes the reduced radius of the contact area by applying two correction factors to adjust both the overestimated contact area and contact time, based on the Batchelor & O'Brien model:

References:

  • A.B. Morris, S. Pannala, Z. Ma, C.M. Hrenya. Development of soft-sphere contact models for thermal heat conduction in granular flows, AIChE Journal, 62(12):4526-4535, 2016

    methods
        %------------------------------------------------------------------
        function this = fixRadius(this,int)
            % Area correction
            % Assumption: time correction negleted (valid for static systems)
            if (~isempty(int.cforcen.total_force))
                fn = norm(int.cforcen.total_force);
            else
                % Assumption: computed with Hertz model
                fn = 4 * sqrt(int.eff_radius) * int.eff_young * int.kinemat.ovlp_n^(3/2) / 3;
            end
            int.kinemat.contact_radius = (fn*int.eff_radius/int.eff_young0)^(1/3);
        end
    end
